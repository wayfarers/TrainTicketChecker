Проект Train Alert

Цей проект розроблений Євгенієм Сушко

Він надає можливість перевіряти наявність квитків на сайті Укрзалізниці та створювати запити на email-поідомлення, коли з'являються нові квитки.

Користувач може створювати запити для напрямку загалом, та на конкретний потяг, із зазначенням необхідного типу місць. Сервіс перевірятиме наявність квитків кожні 15 хвилин.

Використані технології:

  • Spring MVC
  • Spring Data
  • Hibernate
  • MySQL database
  • Quartz
  • AngularJS

Увага:

Сервіс може бути тимчасово недоступний через неочікувані зміни в API Укрзалізниці.
