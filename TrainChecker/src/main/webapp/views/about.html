Train Alert project

This is a project created by Evgeniy Sushko.

It provides an ability to check for available tickets on Ukrzaliznytsia and request email notifications when new places become available.

Users can request notifications about free places for the specified direction or train. The service will check for tickets every 15 minutes.

Used technologies:

  • Spring MVC
  • Spring Data
  • Hibernate
  • MySQL database
  • Quartz
  • AngularJS
